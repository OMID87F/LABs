## 🔹هدف
---
- با **استفاده از** Standard ACL و Extended ACL
	
	- لیست **Standard**:
		- ترافیک خروجی به سمت `1.1.1.1` رو تنها برای شبکه `192.168.10.0/24` باشه.
		- اما IP Address هاست `192.168.30.30` همیشه رد بشه.
		
	- لیست **Extended**: 
		- لپ‌تاپ `02` بتونه با `Web Server 02` ارتباط بگیره ولی فقط **http** باشه.
		- لپ‌تاپ 03 بتونه با` Web Server 03` ارتباط بگیره ولی فقط **https** باشه.
		- بقیه لپ‌تاپ‌ها نتونن به `Web Server`ها وصل بشن.
		- کل شبکه `192.168.1.0/29` بتونه به بقیه شبکه‌ها دسترسی داشته باشه.

- **هدف** از این سناریو اینه که **خیلی کلی** روی هردو ACLهای Standard و Extended **تمرین** کرده باشیم.
